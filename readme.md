# Nordwolf

This is my final, stable and minimal nex configuration. A nord rice - destined in the motto that I should not rice again and focus on the workflow instead. This is sleek, non bloated, Arch + Nix Home-Manager dotfile. As no packages were installed via home-manager except for the themes, the overall system is assumed to be stable(atleast in my case for a long period).

## Install Nex : Yet another tiling window manager for X11. 

>[!NOTE]
> **Nex** is a custom fork of **[bspwm](https://github.com/baskerville/bspwm)**(Binary Space Partition Window Manager). It has all the features of **bspwm** along with some extended features like support for various stack layouts(tall, rtall, wide, rwide, etc)

Steps for Installation : See **[Nex](https://github.com/cobra-r9/nex)**
